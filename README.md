# 🎓 버튼 잡기 게임 (C# 코딩)

## 🎓 25017093 홍시언

## ⭐ 개요
- C# 프로그래밍 학습 (2주차)
- 게임설명 : 도망다니는 버튼을 잡으면서 점수를 얻는 게임입니다. **버튼을 클릭에 성공시, 100점이 추가되며, 도망갈 경우 10점이 감점됩니다.** (점수는 화면 좌측 상단에 나타납니다.)
- 부가설명 : 클릭에 성공할 때마다, 버튼의 크기가 10%씩 감소합니다. 반대로 50번 이상 버튼을 놓치게 될 경우, 게임이 종료됩니다.
- 화면구성 : 아래 스크린샷 참조

## 🏃 실행방법
1. 아래 명령어를 원하는 폴더에 입력합니다. (사전에 git 툴을 설치해두셔야 합니다)
```
git clone https://github.com/suwonhongsi/CatchButton
```
2. Visual Studio를 엽니다.
3. 실행 후, 상단에 초록색 Run버튼을 누른 후, 게임을 재밌게 즐깁니다. 😊😊

## 💡 구현 시 어려웠던점 & 느낀점
- 버튼이 너무 빨리 움직여서 버튼을 잡았을때의 상황을 테스트 하기 쉽지 않았습니다
- 일부 코드들은 Gemini를 참고하여 제작하였습니다. 하지만 생각외로 간단한 코드들인지라 매우 놀랐습니다
- 조금만 더 응용하면은 재밌는 게임으로 만들 수 있지 않을까 싶습니다
- 그 외에도, 제가 수정한 변경사항들을 `git commit`으로 남길 수 있었던 점이 좋았습니다.
